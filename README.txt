易背单词――多功能背单词软件。

【使用说明】
提示：使用前请安装Tkinter, Wordnet, Xmind, Lxml, Selenium, wordcloud, Beautifulsoup4模块、Google浏览器与chromedriver。
步骤一：点击【选项】-【添加】可导入单词文件，包括【从互联网】与【从本地】两种导入方式。其中，从互联网途径导入的文件仅需含有单词，从本地途径导入的文件需含有单词与释义。
步骤二：点击任意按钮开始背单词。根据单词选择正确释义。选择完成后可在空格中手动输入易混淆单词。
步骤三：背单词完成后，可点击【选项】-【更多】生成思维导图（Xmind格式）、单词记忆词云与Excel文件，其中思维导图与Excel文件包含单词释义、近义词、反义词、易混淆词、同根词、词组、例句。
步骤四：背单词完成后，可点击【复习】-【获取文章】获得单词对应ChinaDaily文章页面。
